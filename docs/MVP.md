# MVP Definition

## Minimum Viable Product

### Core Features

1. **Daily Check-ins**
   - Quick daily entries
   - Mood/energy tracking
   - Goal progress updates

2. **Progress Dashboard**
   - Visual progress charts
   - Streak tracking
   - Achievement badges

3. **AI Insights**
   - Pattern recognition
   - Suggestions for improvement

4. **Data Sync**
   - Supabase backend
   - Offline support via RxDB
